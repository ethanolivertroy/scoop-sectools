# Scoop Security Tools Bucket

Scoop bucket for security and compliance CLI tools.

## Installation

```powershell
scoop bucket add sectools https://github.com/ethanolivertroy/scoop-sectools
```

## Available Tools

### cmvp

Terminal UI for searching NIST CMVP validated cryptographic modules.

```powershell
scoop install cmvp
```

### kevs-tui

Terminal UI for browsing CISA Known Exploited Vulnerabilities (KEV) catalog.

```powershell
scoop install kevs-tui
```

### fedramp

Terminal UI for browsing FedRAMP authorized cloud services.

```powershell
scoop install fedramp
```

### vanta-exporter

CLI tool to export Vanta audit evidence organized by control.

```powershell
scoop install vanta-exporter
```
